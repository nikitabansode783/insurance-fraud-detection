# 🚗 Insurance Fraud Detection & Risk Segmentation

## 📌 Overview
This project detects fraudulent insurance claims using Machine Learning and segments customers based on risk.

## 🧠 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Streamlit

## 🔍 Features
- Fraud Detection using Random Forest
- Risk Segmentation
- Interactive UI using Streamlit
- Data Visualization

## 🚀 How to Run
```bash
pip install -r requirements.txt
python -m streamlit run app.py
